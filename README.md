# :car: Car Controller Project

An interactive car animation project built with Vanilla JavaScript.

This project simulates a moving car with animated background effects and keyboard controls.

## :video_game: Controls

- Press Enter to start or stop the car
- Press Space to turn the car lights on/off

## :sparkles: Features

- Smooth car movement animation
- Animated moving background (nature/road effect)
- Background animation pauses without resetting position
- Toggleable car lights
- State management using boolean variables
- Keyboard event handling

## 🛠 Technologies Used

- HTML
- CSS (Animations & Keyframes)
- Vanilla JavaScript (DOM Manipulation & Events)
